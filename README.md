# PPI
Private Program Installer

## Why PPI?
PPI will soon support: .exe, .tar.*, .tar, .deb, .rpm, .appimage.

## Installing on Linux & MacOS
1. Download the file `install.sh` ([click here to download](https://ppi-source.pages.dev/install.sh))
2. In the terminal, run `sudo bash install.sh`
3. You will be guided through the installing process.
